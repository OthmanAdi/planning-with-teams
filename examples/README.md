# Examples

This directory contains example planning files from real team sessions.

## Available Examples

See [skills/planning-with-teams/examples.md](../skills/planning-with-teams/examples.md) for detailed walkthroughs of:

1. **Parallel Code Review** — Security + Performance + Tests reviewers
2. **Debugging with Competing Hypotheses** — Multiple investigators
3. **Feature Development** — Frontend + Backend + Tests
4. **Error Recovery in Teams** — Logging failures to prevent repetition
5. **Research with Multiple Sources** — Parallel information gathering

## Quick Example: Code Review Team

```markdown
# Team Plan: PR #142 Code Review

## Goal
Comprehensive review covering security, performance, and test coverage.

## Team Composition
| Teammate | Role | Focus | Model |
|----------|------|-------|-------|
| Lead | Coordinator | Synthesis | inherit |
| security-reviewer | Reviewer | Vulnerabilities | sonnet |
| perf-reviewer | Reviewer | Performance | sonnet |
| test-reviewer | Reviewer | Test coverage | haiku |
```

## Usage

1. Copy the relevant template from `../templates/`
2. Customize for your specific task
3. Spawn teammates according to the plan
4. Follow Manus principles throughout
